Section A: Multiple Choice Questions (5 x 2 = 10 Marks)

1. The rational number -3/5 lies between:

Answer: a) 0 and -1

-3/5 = -0.6, which is between 0 and -1

2. If the cost price is Rs 400 and selling price is Rs 480, the profit percentage is:

Answer: b) 20%

Profit = 480 - 400 = 80
Profit % = (80/400) × 100 = 20%

3. The sum of the angles of a triangle is:

Answer: b) 180°

Angle sum property of a triangle

4. Simplify: 5x + 3y - 2x + 4y

Answer: b) 3x + 7y

(5x - 2x) + (3y + 4y) = 3x + 7y

5. The mean of 12, 15, 18, 21, 24 is:

Answer: b) 18

Mean = (12+15+18+21+24)/5 = 90/5 = 18

Section B: Short Answer Questions (5 x 2 = 10 Marks)

1. Express -18/24 in its lowest terms. Then find its decimal representation. [2]

-18/24 = -18÷6 / 24÷6 = -3/4

Decimal: -3 ÷ 4 = -0.75

2. A shopkeeper buys a book for Rs 150 and sells it at a loss of 12%. Find the selling price. [2]

Cost Price (CP) = Rs 150

Loss = 12% of 150 = (12/100) × 150 = Rs 18

Selling Price (SP) = CP - Loss = 150 - 18 = Rs 132

3. In a triangle, two angles are 65° and 45°. Find the third angle. What type of triangle is it based on angles? [2]

Sum of angles in a triangle = 180°

Third angle = 180° - 65° - 45° = 70°

Since all angles are less than 90°, it is an acute-angled triangle

4. Subtract: (7a - 3b + 2c) - (4a + 5b - 3c) [2]

= 7a - 3b + 2c - 4a - 5b + 3c

= (7a - 4a) + (-3b - 5b) + (2c + 3c)

= 3a - 8b + 5c

5. Find the median of: 7, 12, 3, 9, 15, 8, 11 [2]

Arranging in ascending order: 3, 7, 8, 9, 11, 12, 15

Number of values = 7 (odd)

Median = middle value = 4th value = 9

2. Profit, Loss, and Percentage: [4]

a) CP = Rs 15,000, Profit = 18%

Profit amount = (18/100) × 15,000 = Rs 2,700

SP = CP + Profit = 15,000 + 2,700 = Rs 17,700

b) SP = Rs 4,500, Loss = 10%

SP = CP - 10% of CP = 90% of CP

4,500 = (90/100) × CP

CP = 4,500 × 100/90 = Rs 5,000

c) Marked Price (MP) = Rs 800, SP = Rs 680

Discount = MP - SP = 800 - 680 = Rs 120

Discount % = (120/800) × 100 = 15%

d) 60% of a number = 144

(60/100) × x = 144

x = 144 × 100/60 = 240

3. Properties of Triangles: [4]

a) Sum of angles = 60° + 80° + 50° = 190°

No, a triangle cannot have these angles because the sum of angles must be exactly 180°.

b) In a right-angled triangle, one angle = 90°

Given acute angle = 35°

Other acute angle = 180° - 90° - 35° = 55°

c) Exterior angle = sum of interior opposite angles

110° = 45° + other angle

Other interior opposite angle = 110° - 45° = 65°

d) In triangle ABC: A + B + C = 180°

2x + 3x + 4x = 180°

9x = 180°

x = 20°

Angle A = 40°, Angle B = 60°, Angle C = 80°

4. Algebraic Expressions: [4]

a) (3x² + 5x - 7) + (2x² - 3x + 4) + (-x² + 2x - 1)

= (3x² + 2x² - x²) + (5x - 3x + 2x) + (-7 + 4 - 1)

= 4x² + 4x - 4

b) (2a + 3b) × (4a - 5b)

= 2a(4a - 5b) + 3b(4a - 5b)

= 8a² - 10ab + 12ab - 15b²

= 8a² + 2ab - 15b²

c) P - Q = (2x² - 3x + 5) - (x² + 4x - 2)

= 2x² - 3x + 5 - x² - 4x + 2

= x² - 7x + 7

d) 3a² - 2ab + b² when a = 2 and b = -3

= 3(2)² - 2(2)(-3) + (-3)²

= 3(4) - 2(-6) + 9

= 12 + 12 + 9 = 33

5. Data Handling - Mean, Median, Mode: [4]

Data: 45, 52, 60, 45, 72, 58, 45, 63, 55, 68

a) Mean = Sum of all values / Number of values

Sum = 45 + 52 + 60 + 45 + 72 + 58 + 45 + 63 + 55 + 68 = 563

Mean = 563/10 = 56.3

b) Arranging in ascending order: 45, 45, 45, 52, 55, 58, 60, 63, 68, 72

n = 10 (even), Median = average of 5th and 6th values

Median = (55 + 58)/2 = 56.5

c) Mode = most frequent value = 45 (appears 3 times)

d) One 45 is replaced by 75

New sum = 563 - 45 + 75 = 593

New mean = 593/10 = 59.3
