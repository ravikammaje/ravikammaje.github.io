ANSWER KEY

Section A: 1) B    2) C    3) C    4) C    5) C

Section B:

1. Photosynthesis is the process by which plants make food using sunlight, water, and carbon dioxide.
2. Speed is the rate of distance covered. Velocity is speed with direction (a vector quantity).
3. Crust, Mantle, Core (inner core and outer core)
4. An atom is the smallest unit of matter that retains chemical properties. Subatomic particles: protons, neutrons, electrons.
5. An object at rest stays at rest, and an object in motion stays in motion at constant velocity, unless acted upon by an external force.

Section C:

1. During inhalation, the diaphragm contracts and lungs expand, drawing air in. Oxygen passes through alveoli into blood. Blood carries oxygen to cells where it is used for energy. Carbon dioxide returns to lungs and is exhaled.
2. (a) Speed = Distance/Time = 150/3 = 50 km/h. (b) Time = Distance/Speed = 250/50 = 5 hours.
3. Physical change: reversible, no new substance formed (e.g., melting ice, tearing paper). Chemical change: irreversible, new substance formed (e.g., burning wood, rusting iron).
4. Accept correctly labeled diagram showing mouth, oesophagus, stomach, small intestine, large intestine in correct order.
5. An electric circuit is a closed loop where current flows from a power source through components and back. Conductors allow current to flow (e.g., copper, aluminum). Insulators block current (e.g., rubber, plastic).
